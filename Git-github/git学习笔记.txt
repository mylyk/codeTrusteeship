使用Github
	
	*************网址：geihub.com
	
	1.目的
		借助github托管项目代码
	2.基本概念
		1）仓库(Repository)
			即你的项目，你想在github上开源一个项目，那就必须要新建一个repository，如果你开源的项目多，你就拥有多个repository。
			用来存放你的项目代码，每个项目对一个项目代码，多个开源项目则有多个仓库。
		2）收藏(star)
			仓库主页的star按钮，意思为收藏项目的人数，在geithub上如果你有一个项目获得100star都算很不容易了！
			收藏项目，方便下次查看
		3）复制和克隆(fork)
			将别人好的项目完完整整的复制过来，且该复制过来的项目是完完整整的独立的。
